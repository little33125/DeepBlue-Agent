# DeepBlue-Agent
DiveAgent: An AI-driven Intelligent Assistant for Scuba Divers and Marine Exploration.

DeepBlue Agent is an open-source AI agent designed to modernize how scuba divers plan, log, and analyze their dives. By integrating LLMs with specialized marine, weather, and dive-safety data, the project automates tedious dive log entry, generates localized dive site safety briefings, and supports marine life identification via computer vision.

Our goal is to enhance underwater safety, streamline dive data management, and leverage "citizen science" to contribute structured, crowd-sourced environmental data to global ocean conservation efforts.
